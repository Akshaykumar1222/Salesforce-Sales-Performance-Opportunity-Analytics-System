# Salesforce Sales Performance & Opportunity Analytics System

## Project Overview
This project is developed as part of the SmartBridge – TCS Last Mile
Salesforce Virtual Internship program.

The objective of this project is to analyze sales performance and manage
opportunities using Salesforce CRM features.

## Business Problem
Sales teams often lack real-time visibility into opportunity pipelines,
leading to missed revenue targets and poor forecasting.

## Solution
Salesforce is used to track opportunities, analyze sales performance,
and automate basic deal updates using CRM tools.

## Key Features
- Opportunity pipeline tracking
- Sales performance analysis
- Automated updates using Apex / Flow
- Reports and dashboards

## Salesforce Components Used
- Standard Objects: Opportunity, Account
- Fields: Amount, Stage, Close Date
- Automation: Apex Trigger / Record-Triggered Flow
- Reports & Dashboards

## Tools & Technologies
- Salesforce Trailhead
- Salesforce Lightning Experience
- Apex (Basics)
